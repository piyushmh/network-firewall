sudo route add -net 140.40.40.0 netmask 255.255.255.0 gw 130.30.30.1

Add default gateway to host
route add default gw {IP-ADDRESS} {INTERFACE-NAME}

See the routing table
sudo route -n
netstat -rn


Assign IP to a host
sudo ifconfig eth0 130.30.30.5 netmask 255.255.255.0 up

/opt/monkey/bin/banana start
